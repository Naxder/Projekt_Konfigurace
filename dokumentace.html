Úvod

Dokumentace vytvořená pro projekt nastavení aplikace v xml a jeho transformace. Projekt je vytvořen pro potřebu předmětu Značkovací jazyky

Popis xsd

XSD je je předpis pro XML. Pro definování rozhrní je tvořeno wsdl, kde jsou použitava xsd pro definici každého rozhraní.

Vypis xsd type

  • string - v xml tagu s tímto typem tagu bude očekáván textový řetězec
  • positiveInteger - v xml tagu s tímto typem bude očekáváno pouze kladné celé číslo
  • text2 - v xml tagu s tímto typem bude očekáván textový řetězec od dnou znacích (UK,CZ,DE...)
  • urlText - v xml tagu s tímto type bude očekávána validní url adrese, dle regularního výrazu

Popis Atributy xml

Atributy často přenášejí informace, které nejsou částí dat. V příkladu dole, typ souboru je vzhledem k datům bezvýznamný, ale mohl by být důležitý pro software, který s elementem manipuluje.
